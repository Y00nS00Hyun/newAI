# 데이터셋 설명

본 대회에서는 총 3개의 데이터셋이 제공됩니다.

## 1. dataset_1.csv
- 규모: 32,470개 샘플
- 구성: title, text, label
- 구분자: 세미콜론(;)
- 평균 텍스트 길이: 약 2,483자
- 라벨 분포: fake(54%) / real(46%)
- 대규모 뉴스 기사 형태의 본문 중심 데이터셋

## 2. dataset_2.csv
- 규모: 250개 샘플
- 구성: title, text, subject, date, label
- 구분자: 쉼표(,)
- 평균 텍스트 길이: 약 2,584자
- 라벨 분포: fake(47%) / real(53%)
- 메타정보(subject, date)를 포함한 정제된 기사형 데이터셋

## 3. dataset_3.csv
- 규모: 250개 샘플
- 구성: text, label
- 구분자: 쉼표(,)
- 평균 텍스트 길이: 약 342자
- 라벨 분포: real(71%) / fake(29%)
- 짧은 문장 중심의 요약형/트윗형 데이터셋

---

### 공통 사항
- label은 `{fake, real}` 두 클래스로 구성되어 있습니다.
- 필요에 따라 dataset_1~3을 병합하거나 개별적으로 사용할 수 있습니다.
- dataset_2의 `subject`, `date`는 선택적으로 feature로 활용할 수 있습니다.
